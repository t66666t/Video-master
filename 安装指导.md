# macOS 应用安装与运行指南

由于本应用目前处于测试阶段，尚未在 Mac App Store 上架，因此在 macOS 上运行时可能会遇到安全提示。请按照以下步骤操作以顺利运行应用。

## 1. 下载与解压

1. 您收到的文件是一个 `.zip` 压缩包（例如 `fluent_player_macos.zip`）。
2. 双击该压缩包进行解压。
3. 解压后，您会看到一个名为 `fluent_player.app` 的文件。
4. **推荐操作**：将 `fluent_player.app` 拖入您的 **「应用程序」 (Applications)** 文件夹中，以便管理。

## 2. 首次运行（关键步骤）

macOS 的安全机制（Gatekeeper）默认会阻止未签名的应用运行。

### ✅ 方法一：使用右键打开（推荐）

这是最简单的方法，只需操作一次。

1. 在 Finder（访达）中找到 `fluent_player.app`。
2. **不要直接双击打开！**
3. **右键点击**（或按住 Control 键点击）该应用图标。
4. 在弹出的菜单中选择 **「打开」**。
5. 系统会弹出一个对话框，提示“无法验证开发者”或“应用来自未识别的开发者”。
6. 此时对话框中会出现 **「打开」** 按钮，点击它即可运行。

> **注意**：以后您就可以像普通应用一样直接双击打开它了。

### 🔧 方法二：系统设置允许

如果方法一无效，请尝试：

1. 双击应用，关闭弹出的错误提示。
2. 打开屏幕左上角的苹果菜单  -> **「系统设置」** (System Settings)。
3. 进入 **「隐私与安全」** (Privacy & Security)。
4. 在右侧向下滚动，找到 **「安全性」** 部分。
5. 您会看到一条提示：“已阻止使用‘fluent_player’，因为来自身份不明的开发者”。
6. 点击旁边的 **「仍要打开」** (Open Anyway) 按钮。
7. 在弹出的确认框中输入您的电脑密码。

### 💻 方法三：终端命令（高级用户）

如果上述方法都无法解决“应用已损坏”的提示，可以使用终端命令清除隔离标记。

1. 打开 **「终端」** (Terminal) 应用（可以在“启动台”->“其他”中找到）。
2. 复制以下命令粘贴到终端中（注意命令最后有一个空格）：
   ```bash
   xattr -cr /Applications/fluent_player.app
   ```
   *(如果您没有将应用放入应用程序文件夹，请将路径替换为您实际存放的路径，或者直接将应用图标拖入终端窗口)*
3. 按下回车键。
4. 现在您可以正常打开应用了。
